## Eclipse快捷键
```
选中代码后，CTRL+ALT+T，添加try catch块。

【Alt+←】、【Alt+→】 
后退历史记录和前进历史记录，在跟踪代码时非常有用。

alt+insert 插入getter和setter方法 
在pom文件中使用快捷键alt+insert 添加依赖
 
shift 按两下 查找类

ctrl + H  全局查找  
ctrl + F  格式化代码  
 
选中某段，Alt+↑  移动这一段  
ctrl + ←/→  按照单词快速移动光标到左右首位处  
shift + ctrl + ←/→  左右箭头选中单词  
 
ctrl + D  快速删除整行
 

Ctrl + H 全局文件内容搜索
Ctrl + Shift + R 全局文件名搜索  
 
Ctrl + Shift + T JAVA类搜索  
  
文件中字符串搜索或替换  
Ctrl + F
 
查看java类的继承结构  
F4
 
查看快捷键或者编辑快捷键  
Ctrl + Shift + L
 
行数搜索  
Ctrl + L
 
psvm  输入main函数
sout 打印
fori 加tab键 创建for循环
```
